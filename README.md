# Классификация выживших на Титанике

## Цель
Провести сравнение классификации с использованием метода ближайшего соседа и логистической регрессии на примере классического датасета о пассажирах на Титанике.

## Стек
 * python
 * sklearn

## Основные этапы
 1. Подготовка данных, разбиение на тестовое и проверочное подмножества.
 2. Классификация логистической регрессией.
 3. Классификация методом ближайшего соседа.
 4. Подбор гиперпараметров.
